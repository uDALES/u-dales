# Fortran API reference

The API documentation of the uDALES Fortran source — modules, procedures, variables, and their call graphs — is generated automatically from comments in the source code using [FORD](https://github.com/Fortran-FOSS-Programmers/ford).

[:octicons-arrow-right-24: Browse the Fortran API reference](https://udales.github.io/u-dales/udales-docs-software/index.html)

This reference is intended for developers working on the solver; for using the model, start from the [workflow overview](udales-workflow.md) instead.

!!! note
    The API reference is built separately from this site (in CI, or locally with `ford ford.md` from the repository root), so the link above points to the published documentation and will not work in a local `mkdocs serve` preview.
