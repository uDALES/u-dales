# Claude Code entry point

Repo guidance for ALL AI agents lives in `AGENTS.md` (the cross-vendor
standard) — read it first. It maps the codebase and routes to the repo-local
skills in `.github/skills/` (`udales-detect`, `udales-exec`, `udales-perf`),
whose `references/` files are the shared, self-updating record of
machine/cluster facts. Do not duplicate guidance here; add it to AGENTS.md
or the skills so every agent product sees it.
